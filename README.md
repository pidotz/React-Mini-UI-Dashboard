**# React-Mini-UI-Dashboard**
A mini dashboard UI built with React as a self-learning project to explore modern frontend development. The dashboard consists of interactive data tiles that display key information, with each tile clickable to navigate to a detailed view. The interface also includes smooth animations and glowing visual effects to create a modern user experience.

**Features**
⚡ Built with React + Vite
🎨 Styled using Tailwind CSS
✨ Smooth animations with Framer Motion
📊 Interactive dashboard with clickable data tiles
🔍 Detailed view for each dashboard tile
🌈 Modern UI with glow effects and hover animations
📱 Responsive layout for desktop and mobile
🛣️ Navigation powered by React Router DOM

**Tech Stack**
React
Vite
Tailwind CSS
Framer Motion
React DOM
React Router DOM

**Purpose**
This project was created to practice building a modern, responsive, and interactive dashboard while gaining hands-on experience with React, Vite, Tailwind CSS, Framer Motion, and React Router DOM.
